React Essentials Assignment

This repository contains two React projects demonstrating core React concepts such as components, props, state management, event handling, conditional rendering, and dynamic lists.

📁 Projects

Part A – Portfolio Card Application

Features:

* React Components
* JSX
* Props
* useState
* Theme Toggle
* Like Counter
* Profile Image Slider
* Contact Alert
* Responsive Design

Folder:

portfolio-card/

Part B – Movie Explorer Application

Features:

* Search Movies
* Filter Local Data
* Conditional Rendering
* Favourite Toggle
* Favourite Movies List
* React State Management

Folder:

movie-explorer/
